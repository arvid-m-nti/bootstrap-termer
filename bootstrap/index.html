HTML

HTML står för HyperText Markup Language. Generellt så visar det upp olika grejer på en webbsida, till exempel texter och bilder.

Elementet HTML är roten till alla andra HTML element.


ID & Class

ID kan bara användas av ett HTML element. Genom att beskriva ett specifikt ID i ett stylesheet så ändras elementet som har det ID till det man ändrade.

Class är likadant som ID fast flera HTML element kan använda det.


Meta

Meta är ursprungligen ett prefix från grekiska som betyder efter eller bakom. Det kan beskrivas som en diskussion om en diskussion, men i sammanhang med webbutveckling är det ungefär en definition.


Responsive

Att en sida är responsive betyder att den anpassas till olika situationer. Till exempel kan det vara att sidan ska se bra ut oberoende av skärmstorleken.


HTTP & HTTPS

HTTP, HyperText Transfer Protocol, används för att visa en sida på internet.

HTTPS, HyperText Transfer Protocol Secure, är som HTTP fast krypterat.
